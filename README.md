# BOMICO HQ UI

Second isolated frontend for the future Mission Control Center.

## Intended URL
- Primary: `hq.bomico.de`
- Optional alias/staging: `mission.bomico.de`
- Existing operational surface remains: `control.bomico.de`

## Purpose
This app is the safer place to build the next-generation BOMICO Mission Control without breaking the current control dashboard.

## Current status
- Bootstrapped from the existing React/Vite dashboard stack
- First Mission Control landing page implemented
- Builds successfully locally
